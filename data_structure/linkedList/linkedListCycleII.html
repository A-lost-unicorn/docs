# 142. 环形链表 II

力扣链接 (opens new window)

给定一个链表的头节点 head ，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。

如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 pos 来表示链表尾连接到链表中的位置 （索引从 0 开始）。如果 pos 是 -1，则在该链表中没有环。注意：pos 不作为参数进行传递，仅仅是为了标识链表的实际情况。

不允许修改 链表。

输入：head = [3,2,0,-4], pos = 1
输出：返回索引为 1 的链表节点
解释：链表中有一个环，其尾部连接到第二个节点。

# 思路

  • 要移动到index.next 只需要判断index不为空
  • 首先判断有没有成环，用快慢指针跑，相遇就是有环。
  • x+n(y+z)+y ---->fast指针走的路径，x+y----->slow指针走的路径
  • fast是slow的两倍，x+n(y+z)+y=2(x+y)，得n(y+z)=x+y ，即x=(n-1)(y+z) +z
  • 设n为1，x=z
  • 所以从交点处开始，与从头开始，一起走
  • 什么时候相遇，什么时候就是环的起点
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode fastIndex = head;
        ListNode slowIndex = head;
        while (fastIndex != null && fastIndex.next != null) {
            fastIndex = fastIndex.next.next;
            slowIndex = slowIndex.next;
            if (fastIndex == slowIndex) {//相遇 有环
                ListNode index1 = fastIndex;
                ListNode index2 = head;
                while (index1 != index2) {//相遇时跳出循环，此时位置为环入口
                    index1 = index1.next;
                    index2 = index2.next;
                }
                return index1;
            }
        } 
        return null;
    }
}
